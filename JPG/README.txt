타로 카드 이미지 넣는 법
============================
이 폴더에 tarotapi.dev와 같은 코드로 파일 이름을 맞춰 주세요.

파일명: {name_short}.jpg  또는  {name_short}.JPG
(앱이 .jpg를 먼저 시도한 뒤 없으면 .JPG를 시도합니다.)

예시 (메이저): ar01.jpg … ar22.jpg
예시 (마이너): wa01.jpg, cuqu.jpg, sw05.jpg, peki.jpg 등
코트: waac(에이스 완즈), wapa, waqu, waki, wakn … 수트 앞두 글자 + 값 약어

전체 목록은 tarotapi.dev 의 /api/v1/cards JSON 의 name_short 필드를 참고하면 됩니다.
